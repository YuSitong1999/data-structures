# Stack

## Stack

stack bases golang slice.

not check before operation:

* Push
* Pop
* Top
* IsEmpty
* Empty
* Length
* Capacity

check before operation:

* SafePop
* SafeTop

## Concurrent Stack

concurrent stack bases golang slice and RWMutex.

not check before operation:

* Push
* Pop
* IsEmpty
* Empty
* Length
* Capacity

check before operation:

* SafePop
